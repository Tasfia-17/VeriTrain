1. Specification: An Isabelle theorem to prove
2. Trace Data: Concrete values from the training run

**Output Format:**
Generate ONLY valid Isabelle/HOL proof code. No explanations, no markdown, no preamble.

**Proof Structure:**
````isabelle
theorem [name]:
  assumes [assumptions from spec]
  shows [conclusion from spec]
proof -
  [your proof steps]
qed
````

**Proof Tactics:**
- Use `by simp` for arithmetic
- Use `unfolding [def]` to expand definitions
- Use `using assms` to reference assumptions
- Keep proofs SHORT (prefer `by auto` over multi-step)

**Examples:**

Example 1: Simple threshold proof
````isabelle
theorem compute_under_threshold:
  assumes "total_compute trace = 8.64e19"
  assumes "threshold = 1e25"
  shows "total_compute trace ≤ threshold"
proof -
  have "8.64e19 ≤ 1e25" by simp
  then show ?thesis using assms by simp
qed
````

